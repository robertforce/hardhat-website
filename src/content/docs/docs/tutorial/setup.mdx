Before we start, make sure you have [Node.js](https://nodejs.org/) and a package manager (such as `npm` or `pnpm`) installed:

<Tabs syncKey="package-manager">
  <TabItem label="npm" icon="npm">
    <Code
      code={`node -v\nnpm -v`}
      lang="sh"
      frame="terminal"
    />
  </TabItem>

  <TabItem label="pnpm" icon="pnpm">
    <Code
      code={`node -v\npnpm -v`}
      lang="sh"
      frame="terminal"
    />
  </TabItem>
  <TabItem label="Yarn" icon="seti:yarn">
    <Code
      code={`node -v\nyarn -v`}
      lang="sh"
      frame="terminal"
    />
  </TabItem>
</Tabs>

Let's start by creating an empty directory and initializing a Hardhat project inside it:

<Tabs syncKey="package-manager">
  <TabItem label="npm" icon="npm">
    <Code
      code={`mkdir hardhat-tutorial && cd hardhat-tutorial\nnpx hardhat --init`}
      lang="sh"
      frame="terminal"
    />
  </TabItem>

  <TabItem label="pnpm" icon="pnpm">
    <Code
      code={`mkdir hardhat-tutorial && cd hardhat-tutorial\npnpm dlx hardhat --init`}
      lang="sh"
      frame="terminal"
    />
  </TabItem>
  <TabItem label="Yarn" icon="seti:yarn">
    <Code
      code={`mkdir hardhat-tutorial && cd hardhat-tutorial\nyarn dlx hardhat --init`}
      lang="sh"
      frame="terminal"
    />
  </TabItem>
</Tabs>

You'll be prompted to select a few options. Use the defaults for everything except the project type:

1. Version: "Hardhat 3 Beta" (default)
2. Directory: current (default)
3. **Project type: "A minimal Hardhat project" (not default)**
4. Install dependencies (default)

Once the setup is complete, verify that everything is working with the following command:

<Run command="hardhat -h" />

You should see Hardhat's help printed in your terminal.

:::tip
Read our [Hardhat projects explanation](/docs/explanations/hardhat-projects) to better understand the structure of a Hardhat project.
:::
