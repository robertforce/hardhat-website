Everything you've done so far uses Hardhat's built-in features. But you can extend Hardhat's functionality with plugins.

Let's install the `hardhat-toolbox-viem` plugin, which adds many features we'll use in the rest of the tutorial.

<Install packages="@nomicfoundation/hardhat-toolbox-viem" />

Import the plugin in your `hardhat.config.ts` file and add it to the list of plugins:

```ts ins={1,5}
import hardhatToolboxViemPlugin from "@nomicfoundation/hardhat-toolbox-viem";
import { defineConfig } from "hardhat/config";

export default defineConfig({
  plugins: [hardhatToolboxViemPlugin],
  solidity: {
    version: "0.8.28",
  },
});
```

Run the help command again to verify that the plugin was loaded:

<Run command="hardhat -h" />

You should see new tasks added by the plugin, like `test nodejs`.
