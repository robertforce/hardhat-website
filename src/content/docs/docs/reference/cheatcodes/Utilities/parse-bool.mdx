### Signature

```solidity
function parseBool(
  string calldata stringifiedValue
) external pure returns (bool parsedValue);
```

### Description

Parses the value of `string` into `bool`

### Examples

```solidity
string memory boolAsString = "false";
bool stringToBool = vm.parseBool(boolAsString); // false
```
