## `Wallet`

### Definiton

```solidity
struct Wallet {
    /// The wallet's address.
    address addr;
    /// The wallet's public key `X`.
    uint256 publicKeyX;
    /// The wallet's public key `Y`.
    uint256 publicKeyY;
    /// The wallet's private key.
    uint256 privateKey;
}
```

### Description

A wallet with a public and private key.
